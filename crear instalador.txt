Resumen para generar el ejecutable e instalador de OpenDataLite

1. Preparación del proyecto
- Verificar que la aplicación funcione correctamente desde main.py.

2. Crear el ejecutable con PyInstaller
Comando usado:
    pyinstaller --name OpenDataLite --onefile --windowed --icon=logo.ico --add-data "logo.ico;." main.py 

Resultado:
- Se generó el archivo dist/OpenDataLite.exe.
- Se usó el ícono logo.ico.
- Se eliminó la consola negra con --windowed.
- Se probó que el .exe funcione correctamente.

3. Crear el instalador con Inno Setup 
- Descarga del inno Setup (https://jrsoftware.org/isdl.php#stable)
- Se creó un script .iss (ej: OpenDataLite_instalador.iss), se agrega al proyecto.
- Se configuró:
    - Ruta del ejecutable: dist\OpenDataLite.exe
    - Ícono del instalador: SetupIconFile=logo.ico
    - Nombre del instalador: OpenDataLite_Instalador.exe
    - Accesos directos y carpeta de instalación
